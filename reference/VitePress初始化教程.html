安装VitePress ​

  • 新建一个项目文件夹，使用 npm add -D vitepress 进行配置

    • npm add -D 安装的依赖只用于当前文件夹下的项目 它会安装到当前项目的 node_modules 中，不会影响其他项 目。

    • 仅作为开发时使用 这些包通常用于开发阶段（如测试、编译、打包工具等），生产 环境不需要(编译打包时，只有代码中引入的依赖包才会被一并打包)。

    • 其他文件夹想使用必须再次安装 每个项目的依赖都是独立的，换一个项目就要重新安装。

  • 执行初始化命令，让VitePress帮我们搭建一个基础的框架
    vitepress init
